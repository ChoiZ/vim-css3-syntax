TITLE
=====

vim-css3-syntax


DESCRIPTION
===========

Add CSS3 syntax support to vim's built-in `syntax/css.vim`.


INSTALLATION
============

Extract vim-css3-syntax.tar.gz and copy contents to `~/.vim` directory.


Git and pathogen
================

    $ cd ~/.vim/bundle
    $ git clone https://hail2u@github.com/hail2u/vim-css3-syntax.git
